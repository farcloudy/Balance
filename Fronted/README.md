# Balance 前端

基于 Vue 3 + Vite + Pinia 的记账系统前端。

## 技术栈

- Vue 3
- Vite
- Pinia (状态管理)
- Vue Router
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
src/
├── api/          # API 接口
├── assets/       # 静态资源
├── components/   # 公共组件
├── router/       # 路由配置
├── stores/       # Pinia 状态管理
├── utils/        # 工具函数
├── views/        # 页面组件
├── App.vue       # 根组件
└── main.js       # 入口文件
```

## 代码规范

- 使用 4 个空格缩进
- 单引号
- 不使用分号
